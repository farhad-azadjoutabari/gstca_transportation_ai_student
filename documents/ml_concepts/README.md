## Quick Concept Map

Machine learning is a way for a computer system to learn patterns from examples.

- A model is the learned pattern.
- Features are the input columns.
- A label is the answer column, if one exists.
- Training is the process of learning from data.
- Inference is using a trained model to make a prediction or generate output.
- Supervised learning uses labels.
- Unsupervised learning does not use labels.
- Semi-supervised learning uses a small amount of labeled data plus a larger amount of unlabeled data.
- Reinforcement learning learns through actions, rewards, and feedback.
- Generative AI creates new content, such as text, images, audio, code, or synthetic data.
